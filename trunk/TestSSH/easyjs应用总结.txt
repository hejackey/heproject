一、easyui.js form表单使用
1、input等元素class必须定义成 class="easyui-validatebox"

2、非空验证
   给input元素加required属性true，required="true",默认easyui.js中有英文错误提示，可给input添加missingMessage和invalidMessage属性，
   分别指定：当文本框为空时提示的文本信息、当文本框内容不合法时提示的文本信息。直接覆盖js中原有的提示信息
   
3、默认支持的validType类型：email、url、length[param1,param2]，可require和validType组合起来使用

4、扩展自定义validType
	 $.extend($.fn.validatebox.defaults.rules, {
		    mobile: {
		        validator: function(value){
		        	return /^(1\d{10})$/.test(value);
		        },
		        message: '请输入正确的手机号.'
		    }
		});
